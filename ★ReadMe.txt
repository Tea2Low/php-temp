不可視ディレクトリとして「.vscode」があり、中にVScode用の「settings.json」が入っているので注意。
●command + Shift + 「.」（ピリオド）……不可視ファイルが表示される。

「settings.json」には、PHP用にカスタマイズしたプロキシ設定が入っている。
●"enable": true,
●"proxyUri": "http://192.168.10.40:8888/"
※「proxyUri」はPCによって変わる
